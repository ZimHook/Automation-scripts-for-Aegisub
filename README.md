    | C Font Resize (Mocha Deshaking) | v1.1    |
    | C Gradient                      | v2.0    |

* 使用方法 \
    + 将LUA脚本复制到`C:\Program Files (x86)\Aegisub\automation\autoload`路径下，或你的Aegisub安装相应位置
    + 在Aegisub Automation项中可以发现添加的脚本
